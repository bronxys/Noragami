## Instalação Via Termux  <img src="https://user-images.githubusercontent.com/108157095/182052725-6568419a-6a9f-490a-85ea-90b94af694fe.png" height="25px">
**1° Comando**
```
pkg install git -y && pkg install wget -y && pkg install nodejs -y && pkg update -y && pkg upgrade -y && pkg install nodejs-lts -y && pkg install ffmpeg -y && apt install -y ffmpeg nodejs wget tesseract git && apt-get upgrade && apt-get update
```
**2° Comando**
```
termux-setup-storage
```
**3° Comando**
```
cd /sdcard && git clone https://github.com/048Regis/noragami
```
**4° Comando**
```
cd noragami && sh start.sh
```
